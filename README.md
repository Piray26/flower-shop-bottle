# 🌸 FlowerMaster - Интернет-магазин цветов

Веб-приложение для продажи цветов с возможностью сборки собственного букета.

## 📋 О проекте

FlowerMaster - это интернет-магазин цветов, разработанный в рамках лабораторной работы. Проект включает каталог букетов, конструктор букетов, корзину и контактную форму.

### Основной функционал:
- 📱 Просмотр каталога букетов
- 🛒 Добавление товаров в корзину
- 🏗️ Конструктор букетов (в разработке)
- 📞 Форма обратной связи
- ℹ️ Страница "О нас" с информацией о компании

## 🛠️ Технологии

### Backend:
- **Python 3.x**
- **Bottle** - микрофреймворк
- **Bottle SimpleTemplate Engine** - шаблонизатор

### Frontend:
- **HTML5/CSS3**
- **Bootstrap** - адаптивный дизайн
- **Montserrat** - шрифты
- **jQuery** - JavaScript библиотека
